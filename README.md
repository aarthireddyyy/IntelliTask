# IntelliTask Orchestrator

A multi-agent workflow system built using CrewAI and a fully local LLM (Ollama).

## Current Capabilities
- Manager agent plans tasks
- Research agent gathers competitor data
- Analyst agent compares competitors
- Writer agent produces a structured report

## Tech Stack
- CrewAI
- Ollama (Qwen 2.5)
- Python 3.10+

## Status
Phase 1 complete: core multi-agent orchestration working end-to-end.
